# 微信官方 js-sdk

基于CI将官方 js-sdk 发布到 npm

> 版本记录于.versionrc
> 文档: https://developers.weixin.qq.com/doc/service/guide/h5/

## 安装

``` bash
pnpm add wechat-open-js-sdk
```

## 使用

``` javascript
// commonjs
var wx = require('wechat-open-js-sdk');

// es module
import wx from 'wechat-open-js-sdk'
```
